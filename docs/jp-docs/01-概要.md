# 01 概要  
Virtualization & Network Simulation Lab

このプロジェクトは、VirtualBox と Cisco Packet Tracer の 2 つの環境を組み合わせて、小規模な企業ネットワークを概念的に再現するラボです。両環境は技術的には接続されていませんが、同時に構築することでネットワークの異なるレイヤーを表現し、1 つの総合的な学習プロジェクトとして機能します。

VirtualBox 環境では、仮想化の基礎、pfSense ファイアウォールの導入、RHEL サーバーの構築を行います。Packet Tracer 環境では、Cisco ルーター／スイッチの設定、LAN トポロジー設計、ルーティング動作のシミュレーションを行います。

これら 2 つのラボを組み合わせることで、仮想化・ファイアウォール・ルーティング・スイッチング・ネットワーク設計・ドキュメント作成といった幅広いスキルを実践的に習得できます。

---

## ラボの目的

- VirtualBox を用いた再現性のある仮想化環境の構築  
- pfSense ファイアウォールのインストールと設定  
- RHEL サーバーの構築と基本設定  
- Cisco Packet Tracer を用いたネットワークトポロジーの設計  
- ルーティング、スイッチング、IP アドレッシングの理解  
- 英日バイリンガルでの技術ドキュメント作成  

---

## なぜ 2 つの環境を使用するのか？

VirtualBox と Packet Tracer は役割が異なります。

- **VirtualBox**  
  - 実際の OS を動作させる  
  - pfSense による本物のファイアウォール挙動  
  - NIC やルーティングテーブルの実機に近い動作  

- **Packet Tracer**  
  - Cisco ネットワークの視覚的シミュレーション  
  - VLAN、STP、ルーティングプロトコルなどの学習  
  - 物理機器なしで高速にトポロジーを構築  

両方を同時に扱うことで、現場で求められる幅広いインフラ知識を実践的に習得できます。

---

## 習得できるスキル

- 仮想マシンの構築  
- VirtualBox ネットワーク設定（NAT / Host-Only）  
- pfSense のインストールとインターフェース設定  
- RHEL サーバーの構築  
- Cisco ルーティング／スイッチングの基礎  
- IP アドレッシングとサブネット設計  
- ネットワークトポロジー設計  
- 英日バイリンガルでの技術文書作成
