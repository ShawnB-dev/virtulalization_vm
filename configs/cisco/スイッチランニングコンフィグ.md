# スイッチランニングコンフィグ

**English version →** [switch-running-config.md](../cisco/switch-running-config.md)


Cisco Packet Tracer – スイッチ設定リファレンス

このドキュメントでは、本ラボで使用している Packet Tracer スイッチの実際のランニングコンフィグを掲載しています。  
VLAN セグメンテーションとルーターへのトランク接続を提供します。

---

## デバイス概要

- **ホスト名:** Switch1  
- **VLAN:**  
  - 10（SERVERS）  
  - 20（CLIENTS）  
  - 30（MGMT）  
- **トランクポート:** G0/1  
- **アクセスポート:** G0/2, G0/3, G0/4  

---

## ランニングコンフィグ（実際の設定）
```markdown
hostname Switch1
vlan 10 name SERVERS
vlan 20 name CLIENTS
vlan 30 name MGMT
interface GigabitEthernet0/1 switchport mode trunk switchport trunk allowed vlan 10,20,30
interface GigabitEthernet0/2 switchport mode access switchport access vlan 10
interface GigabitEthernet0/3 switchport mode access switchport access vlan 20
interface GigabitEthernet0/4 switchport mode access switchport access vlan 30
```

---

## 備考

- G0/1 は Router1 へのトランクアップリンクです。  
- 各アクセスポートは VLAN 10/20/30 に対応しています。  
- VLAN 名は一般的な企業ネットワークの命名規則に合わせています。
