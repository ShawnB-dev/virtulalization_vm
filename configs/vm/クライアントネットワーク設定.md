# クライアントネットワーク設定  

**English version →** [client-network-config.md](../vm/client-network-config.md)

Packet Tracer クライアントホストのネットワーク構成

このドキュメントでは、Packet Tracer 内のクライアント PC に設定されているネットワーク情報をまとめています。

---

## PC1 の設定

| 項目 | 値 |
|------|------|
| IP アドレス | `192.168.1.10` |
| サブネットマスク | `255.255.255.0` |
| デフォルトゲートウェイ | `192.168.1.1` |
| DNS サーバー | （任意）`8.8.8.8` |

---

## PC2 の設定

| 項目 | 値 |
|------|------|
| IP アドレス | `192.168.1.11` |
| サブネットマスク | `255.255.255.0` |
| デフォルトゲートウェイ | `192.168.1.1` |
| DNS サーバー | （任意）`8.8.8.8` |

---

## 備考

- すべてのクライアントは VLAN 1 に属しています（特に変更しない限り）。  
- ゲートウェイ（`192.168.1.1`）は Packet Tracer のルーターが提供します。  
- インターネット接続をシミュレートしない場合、DNS は必須ではありません。
